# Esbuild Plugins Kit

## A collection of essential and useful plugin for esbuild

## Plugins

### [`detect-circular-plugin`](./packages/detect-circular/README.md)

A plugin for esbuild that detects circular dependencies.

### [`html-auto-include`](./packages/html-auto-include/README.md)

Get the HTML entrypoint of your application and add the generated files as link or scripts tags.
